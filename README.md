***小知识：
常见的file操作模式：

    read 打开&读取
    –r：打开指定文件，只用于reading。文件的指针在开头。python的默认模式。若无指定文件则报错
    –·rb：以二进制执行的r；

    write 打开&覆盖
    – w：打开指定文件，只用于writing。如果文件存在，则先删除（表里所有的）已有数据，如果不存在，则创建；
    – wb：以二进制执行的w；

    append 打开&添加
    – a：打开指定文件，用于appending。如果文件存在，指针放在结尾，如果文件不存在，则创建；
    –ab：以二进制执行的a
